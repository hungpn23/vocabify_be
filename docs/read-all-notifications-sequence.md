# Sequence Diagram: Đánh dấu tất cả thông báo đã đọc

Sơ đồ dưới đây mô tả ngắn gọn nghiệp vụ đánh dấu tất cả thông báo chưa đọc của người dùng thành đã đọc.

```mermaid
---

  theme: base
  look: classic
  sequence:
    showSequenceNumbers: false
    mirrorActors: false
    diagramMarginX: 12
    diagramMarginY: 8
    actorMargin: 34
    boxMargin: 6
    boxTextMargin: 4
    noteMargin: 6
    messageMargin: 24
  themeVariables:
    background: "#ffffff"
    mainBkg: "#ffffff"

    signalTextColor: "#0f172a"

sequenceDiagram
	actor User
	participant FE as FE
	participant BE as API
	participant DB as DB
	participant Redis as Redis

	Note over User,Redis: Luồng đánh dấu tất cả thông báo đã đọc
	User->>FE: Chọn đánh dấu tất cả đã đọc
	FE->>BE: Mark all read + access token
	BE->>BE: Verify access token
	BE->>Redis: Check session
	alt Token lỗi hoặc session không tồn tại
		BE-->>FE: Báo không được phép truy cập
		FE-->>User: Yêu cầu đăng nhập lại
	else Phiên hợp lệ
		BE->>DB: Cập nhật mọi unread notifications
		BE-->>FE: Thành công
		FE-->>User: Tất cả đã được đọc
	end
```
